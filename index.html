YNA 클라이언트 보조 도구입니다. AI와 질의응답, AI를 통한 키워드 추천,

다운로드
버전 3.1.0.3

활용법

연합뉴스 작성기 도우미 사용 모습입니다

주요 기능

🔍

빠른 검색

단축키로 작성기를 떠나지 않고 연합뉴스(==)·구글(//) 검색이 가능합니다

🤖

AI 통합

단축키(Ctrl+Alt+Q)로 AI에게 질문해 답변을 받거나 키워드를 추천(Ctrl+Alt+K)받을 수 있습니다. DeepL을 이용한 번역 기능(Ctrl+Alt+H)도 있습니다

🔢

연합뉴스 숫자 표기 적용

아라비아 숫자를 연합뉴스 표기에 맞게 변환(Ctrl+Shift+R)합니다. 실시간 환율을 적용해 외화에 원화 표기를 추가(Ctrl+Alt+R)할 수도 있습니다

✍️

바이라인 자동 입력 등

바이라인을 자동으로 입력(Ctrl+Alt+V)하거나 국어사전 검색(Ctrl+Alt+D) 등도 가능합니다

⚡

업데이트

온라인으로 새 버전이 있는지 확인하고 안내합니다

상세 사용법

📋 주요 기능

기본 기능

  • 송고 버튼 클릭: Ctrl+Alt+P
  • 제목 길이 변경 활성화: Ctrl+Alt+T
  • 통합검색 열기: Ctrl+Alt+F
  • 입력창 순차 선택: Ctrl+Alt+E (내용→속성→키워드)

검색 및 입력 도우미

  • 국어 사전 검색: Ctrl+Alt+D (블록 설정한 단어)
  • 추천분류/키워드 자동 입력: Ctrl+Shift+K
  • 제목 글자 수 셈하기: Ctrl+Alt+C
  • 제목 글자 수 줄이기: Ctrl+Alt+S (LLM API Key 필요)

AI 및 번역

  • 키워드 추출: Ctrl+Alt+K (LLM API Key 필요)
  • 바이라인 자동 입력: Ctrl+Alt+V
  • DeepL 번역: Ctrl+Alt+H (DeepL API Key 필요)

숫자 및 텍스트 처리

  • 연합뉴스 숫자 표기: Shift+Alt+R
  • 외화 원화 표기 추가: Ctrl+Alt+R
  • 텍스트 대치: km→㎞, ~→∼ 등

작성기 내 검색

  • 연합뉴스 검색: == 입력
  • 구글 검색: // 입력

⚙️ 설정

트레이 아이콘(화면 오른쪽 하단 시계 옆 ^ 표시를 클릭하면 나옵니다. 푸른색 배경에 흰색 Y)을 마우스 오른쪽으로 클릭하면 설정 메뉴가 나타납니다.

⚠️ 확인한 문제점

텍스트 대치 기능

원문이 한글이면 작동하지 않습니다. '연합'을 'dusgkq'으로 입력해 사용하는 우회적인 방법이 있으나, 제대로 작동하지 않으니 지양하시길 바랍니다.

프로그램 호환성

기본적으로 YnaClientEx에 한정해 작동되도록 짜여져 있으나, 이외 프로그램에서도 작동되기도 합니다. 향후 수정 예정입니다.
