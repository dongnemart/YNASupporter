활용법

연합뉴스 작성기 도우미 사용 모습입니다

✨ New

메모보드

하나의 창에 텍스트와 이미지를 모아 두고, AI 분석까지 받을 수 있는 메모판입니다

📋

메모판 다중 실행

여러 메모판을 동시에 띄울 수 있으며, 트레이 메뉴에서 새 메모판을 추가로 열 수 있습니다

Ctrl+Alt+A
✏️

텍스트 메모 추가·수정

메모 추가 버튼으로 직접 입력하고, 더블클릭으로 수정할 수 있습니다. 복사·삭제도 가능합니다

📎

클립보드 메모 수집

텍스트 클립보드 감지를 켜면 복사한 텍스트가 자동으로 메모에 추가됩니다. Ctrl+V로 즉시 가져올 수도 있습니다

⏱️

시간에 따라 메모 자동 관리

기본 6시간 보존, 1~24시간 범위로 보존 시간 설정이 가능합니다. ‘메모보드 유지’를 켜면 만료 없이 계속 유지됩니다

💾

저장 & 불러오기

메모판 상태를 JSON으로 저장하고 복원할 수 있습니다. 창 크기·위치·색상·AI 분석 결과까지 함께 저장됩니다

🎨

색상 & 레이아웃

보드 전체·개별 메모 카드 색상을 바꿀 수 있습니다. 창 너비에 따라 메모 카드가 자동 재배치됩니다

🔔

트레이 연동

창을 닫으면 트레이로 숨기고, 복원 단축키(기본 Ctrl+Alt+1~5)로 빠르게 다시 불러올 수 있습니다

🤖

AI 분석

모인 메모 내용을 바탕으로 AI 분석을 요청하면, 결과가 보드 맨 위에 카드로 표시됩니다. 프롬프트도 수정 가능합니다

주요 기능

🤖

AI 통합

단축키(Ctrl+Alt+Q)로 AI에게 질문해 답변을 받거나 키워드를 추천(Ctrl+Alt+K)받을 수 있습니다. DeepL을 이용한 번역 기능(Ctrl+Alt+H)도 있습니다

🧮

AI 퍼센트 계산기

비율값·비율·증감률·증감값 등 4가지 퍼센트 계산을 지원합니다. 텍스트를 선택하면 AI가 기준값과 비교값을 자동으로 추출해 즉시 계산합니다

⌨️

텍스트 자동 대치

자주 쓰는 단어를 여러번 입력하지 않도록 입력어를 타이핑하는 즉시 정해진 단어로 변환합니다. 대치어 설정 창에서 대상 단어를 추가·수정할 수 있습니다.

📈

주가 검색

트레이 메뉴나 단축키(Ctrl+Alt+M)로 작은 창을 띄워 실시간 주가를 조회하고 복사할 수 있습니다

⚡

이미지 리사이즈

사용자가 정한 용량에 맞춰 이미지 크기를 조정합니다

🖼️

사진 콜라주 제작

여러 장의 사진을 하나로 합쳐줍니다. 다양한 레이아웃에 맞춰 2~6장의 이미지를 합칠 수 있습니다.

🔢

연합뉴스 숫자 표기 적용

아라비아 숫자를 연합뉴스 표기에 맞게 변환(Ctrl+Shift+R)합니다. 실시간 환율을 적용해 외화에 원화 표기를 추가(Ctrl+Alt+R)할 수도 있습니다

🔍

빠른 검색

단축키로 작성기를 떠나지 않고 연합뉴스(==)·구글(//) 검색이 가능합니다

상세 사용법

기본 기능

  • 송고 버튼 클릭: Ctrl+Alt+P
  • 제목 길이 변경 활성화: Ctrl+Alt+T
  • 제목 길이 변경 항상 켜기: 설정에서 '제목 길이 변경 항상 켜기' 체크
  • 통합검색 열기: Ctrl+Alt+F
  • 입력창 순차 선택: Ctrl+Alt+E (내용→속성→키워드)

검색 및 입력 도우미

  • 국어 사전 검색: Ctrl+Alt+D (블록 설정한 단어)
  • 추천분류/키워드 자동 입력: Ctrl+Shift+K
  • 바이라인 자동 입력: Ctrl+Alt+V
  • 제목 글자 수 셈하기: Ctrl+Alt+C
  • 제목 글자 수 줄이기: Ctrl+Alt+S (LLM API Key 필요)
  • 메모보드: Ctrl+Alt+A

AI 및 번역

  • 키워드 추출: Ctrl+Alt+K (LLM API Key 필요)
  • 퍼센트 계산기: Ctrl+Alt+N — 텍스트 선택 시 AI가 기준값·비교값을 자동 추출하여 계산 (LLM API Key 필요)
  • DeepL 번역: Ctrl+Alt+H (DeepL API Key 필요)

숫자 및 텍스트 처리

  • 연합뉴스 숫자 표기: Ctrl+Shift+R
  • 외화 원화 표기 추가: Ctrl+Alt+R
  • 텍스트 대치: km→㎞, ~→∼ 등 즉시 반환. 트레이 메뉴에서 '대치어 설정'에서 단어 관리
  • 괄호/따옴표 자동 닫기: (, [, {, <, ", ' 입력 시 해당 기호 자동 닫힘. 설정에서 괄호/따옴표 자동 닫기 켜기

작성기 내 검색

  • 연합뉴스 검색: == 입력
  • 구글 검색: // 입력
  • 주가 검색: Ctrl+Alt+M

⚙️ 설정

트레이 아이콘(화면 오른쪽 하단 시계 옆 ^ 표시를 클릭하면 나옵니다. 푸른색 배경에 흰색 Y)을 마우스 오른쪽으로 클릭하면 설정 메뉴가 나타납니다.

⚠️ 프로그램 호환성

기본적으로 YnaClinetEx에 한정해 작동되도록 짜였으나, 이외 프로그램에서도 작동되기도 합니다.
